Embed a Hidden Message

Upload a carrier file and a secret message. The result will be posted publicly.

Any format: JPG, PNG, MP4, WAV, PDF, TXT, etc.

Use this to skip file headers (e.g. 64 for some JPGs).

Single int (e.g. 8) or cycling list (e.g. 8,16,28).

How it works

The carrier bits are indexed from 0. The first S bits are left untouched (to protect file format headers). Then, starting at bit S, every L-th bit is overwritten with the next bit from your message (with a 32-bit length header prepended automatically). If L is a list, it cycles: 8, 16, 28, 8, 16, 28, … The recipient uses the same S and L to extract the message.
